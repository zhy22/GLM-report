# Chess Win Probability Model

A logistic regression model for predicting chess game outcomes using
game-level attributes scraped from chess.com, developed as a group
assignment for a statistical modelling course at the Australian National
University.

---

## Project Overview

This project builds a Bernoulli logistic regression model that estimates
the probability a chess player wins an individual game. The model is
constructed from over 16,000 scraped games across four titled chess.com
players and is applied to predict the team's winning chances against
Women's World Classical Chess Champion Ju Wenjun.

The modelling pipeline covers data scraping, wrangling, exploratory
analysis, two-stage model selection, diagnostics, and prediction with
uncertainty quantification.
